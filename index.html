NAME

steamcmddl - Download windows versions of Steam games

SYNOPSIS

steamcmddl OPTIONS

DESCRIPTION

This program downloads the Windows version of Steam games. Downloaded programs appear in the ~/steamcmddl folder.

OPTIONS

All options are optional, and don't need to be entered.

-u value
Specify the username beforehand.
-a value
Specify the APPID beforehand.

BUGS

  • SteamCMD occasionally segfaults. When this happens, just rerun the program.

DOWNLOAD

On Ubuntu:

Add the PPA ppa:ev1l0rd/steamcmddl to your PPAs and then install the steamcmddl package.

On Debian-based with steamcmddl in your repositories:

Go to the latest release page and download and install the deb.

On Debian-based without steamcmddl in your repositories:

Go to the 1.2 release page and download and install the deb file.

All other systems:

Download this file and put it in your PATH. Note that this does not include any of the manpages that come with the other packages.
